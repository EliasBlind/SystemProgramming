## Задание 2: Проверка утверждений

Определение истинности высказываний о работе анализатора:

1. **Верно.** Статический анализ — эффективный вспомогательный инструмент.
2. **Неверно.** По умолчанию `cppcheck` анализирует файлы, переданные в аргументах, а не всю текущую директорию (если она не указана).
3. **Верно.** Флаг `-v` или `--verbose` увеличивает количество выводимой информации.
4. **Неверно.** Для использования стандарта C++14 используется флаг `--std=c++14`.
5. **Верно.** Анализатор может игнорировать определенные директории при наличии соответствующих настроек или флагов исключения.
